🎨 Design Guidelines — High-End Editorial Nature & Travel
🌿 Core Concept (Mandatory)
Style: Premium editorial travel & nature experience
Feel: luxurious, immersive, organic, non-rigid
Inspiration: high-end travel magazine (NOT a dashboard, NOT a generic app)
Key Rules:
Avoid rigid or repetitive grid layouts
Use intentional asymmetry
Allow element overlapping
Create depth using layers (NOT borders)
🎨 Color Palette (use exactly these values)
Primary (Leaf Green): #4de082
Background base: #0e1511
Surface lowest: #09100c
Surface highest: #2f3632
Usage:
Main background: #0e1511
Elevated elements: #2f3632
Primary actions: #4de082
🚫 Strict Color Rules
❌ Do NOT use pure black #000000
❌ Do NOT use pure white #ffffff
❌ Do NOT use solid borders (border: 1px solid)
Instead:
Use tonal differences (background shifts)
Use opacity
Use layering
✨ Glassmorphism & Effects

Use glassmorphism for floating elements:

backdrop-filter: blur(20px);
opacity: 0.6 - 0.8;
Apply to:
Navigation bar
Floating containers
Overlays
🧠 Typography
Fonts:
Headlines: Noto Serif
Body text: Manrope
Rules:
Large, expressive headlines
Strong contrast between sizes
Generous line spacing
large display → small body text
🌑 Depth & Elevation

Avoid standard drop shadows.

Instead:
Use tonal layering
Stack different surface levels
If shadow is needed:
blur: 40px - 60px;
opacity: 0.04 - 0.08;
color: dark green (NOT black)
🧩 Components (Styling Rules)
Buttons
Shape: fully rounded (border-radius: 9999px)

Primary:

Background: #4de082

Secondary:

No solid fill
Soft border with opacity
Cards
❌ No divider lines
✅ Use spacing to separate content
Border radius: rounded-xl (~1.5rem)

Images:

Always apply a subtle overlay for readability
Navigation Bar
Style: glassmorphism
Blur: ~24px
Opacity: ~70%

Layout:

Logo → left
Links → center
CTA → right
Chips (Filters)
Default: dark surface tones
Active: use primary color
📐 Layout & Composition
Core Rules:
Use generous vertical spacing (space = luxury)
Avoid perfect symmetry
Allow elements to:
overlap
cross sections
break container boundaries
Examples:
Images bleeding behind text
Buttons floating between sections
Typography bridging different areas
🧠 Design Philosophy
Think in layers, not boxes
Think editorial flow, not technical layout
Design like a magazine, not an app
✅ Best Practices
Use plenty of whitespace
Maintain visual consistency
Create strong hierarchy
Prioritize readability
❌ Avoid Completely
Dashboard-style layouts
Repetitive grids
Visible borders
Generic UI patterns
Crowded elements
Tight typography
🎯 Final Goal

The UI must feel like:

A premium editorial experience
A visually immersive environment
A modern yet organic interface

It must NOT feel like:

A standard web app
A template
A dashboard
